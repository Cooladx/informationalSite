# informationalSite


This project is meant to be able to serve various html files thorugh different URLs in the form of a server running. 

It contains a very basic informational site that contains 4 pages: index, about, contact-me and 404 page.

## Running the server


Run `npm install` to get all packages in the project. 

Then, start up the server by running the command:

`node server.js`
